TSF TAsk 1: 
📊 Student Score Prediction using Machine Learning
🔍 Overview

This project applies supervised machine learning (Linear Regression) to predict student scores based on the number of study hours.

🎯 Objective

To build a predictive model that estimates student performance and demonstrates the relationship between study time and academic results.

📁 Dataset
Contains 25 observations
Features:
Hours (independent variable)
Scores (target variable)

🛠️ Tools & Technologies
Python
Pandas, NumPy
Matplotlib
Scikit-learn

📈 Approach
Data preprocessing and exploration
Visualization of Hours vs Scores
Train-test split (80/20)
Model training using Linear Regression
Performance evaluation using Mean Absolute Error

📊 Results
Model achieved MAE ≈ 4.18
Example prediction:
Study Hours: 9.25
Predicted Score: ~93.7%

💡 Key Learnings
Understanding regression modeling
Model evaluation techniques
Data visualization and interpretation
